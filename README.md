# AI Agent Startup Generator

Multi-agent system using AutoGen that dynamically creates AI agents which collaborate to generate startup ideas.

## Features

- Dynamic AI agent generation
- Multi-agent collaboration
- Async runtime execution
- Startup idea generation

## Setup

Install dependencies:

pip install -r requirements.txt

Create `.env` file:

OPENAI_API_KEY=your_api_key_here

## Run

python main.py
